# 《西行路》游戏设计文档

---

## 一、游戏概述

### 1.1 概念定义

**《西行路》** 是一款2D横版融合中国传统神话西游世界观与Roguelike卡牌构筑机制的创新之作。玩家将扮演不同的取经人，在九九八十一难的不同关卡中击败敌人取得真经。

### 1.2 设计支柱风格

采用动态水墨渲染技术，所有卡牌、角色、场景均以中国山水画风格呈现，配合卷轴式UI界面，营造沉浸式东方神话氛围。

### 1.3 设计支柱

| 设计支柱 | 描述 |
|----------|------|
| **策略深度** | 核心战斗要求玩家在资源、位置、时机之间做出权衡，无绝对最优解 |
| **视觉沉浸** | 水墨美术风格贯穿始终，UI动效传递东方美学的写意与留白 |
| **节奏张力** | 紧张战斗与休闲探索交叉出现，张弛有度的叙事节奏 |
| **重玩价值** | 随机性卡牌池、多种路线选择、保证每次游玩体验独特 |
| **情感共鸣** | 西游故事再诠释，经典角色新演绎，引发玩家文化认同与情感投射 |

### 1.4 目标用户画像

| 维度 | 描述 |
|------|------|
| **核心用户** | 喜爱《杀戮尖塔》《邪恶小屋》等卡牌Roguelike的重度玩家，年龄18-35岁 |
| **扩展用户** | 对中国神话感兴趣的泛用户，偏好策略与角色扮演元素 |
| **潜在用户** | 休闲卡牌玩家，通过简化模式或辅助系统可上手 |
| **用户特征** | 追求策略深度、喜爱收集与构筑、享受反复挑战的成就感 |

## 二、核心创新机制


### 2.1地图结构
采用「九九八十一难」的分支地图设计，分为九大劫难区域：

| 区域 | 劫难类型 | 代表敌人 | 
|------|----------|----------|
| **两界山** | 斩妖 | 猛虎精、熊罴怪 |
| **五行山** | 悟道 | 心魔、五行使者 |
| **高老庄** | 渡人 | 猪妖、狐妖 |
| **白骨岭** | 幻灭 | 白骨夫人、幻影 |
| **火焰山** | 涅槃 | 牛魔王、红孩儿 |
| **盘丝洞** | 迷障 | 蜘蛛精、蜈蚣精 |
| **狮驼岭** | 降魔 | 三魔头、群妖 |
| **凌云渡** | 成佛 | 如来化身、金蝉残魂 |

#### 路线选择
每个区域包含多个分支节点，玩家需选择前进方向：
- **普通遭遇**：战斗、奖励、事件
- **精英战斗**：击败后获得稀有卡牌
- **商店**：消耗金币购买/出售卡牌
- **篝火休息**：恢复生命或升级卡牌
- **神秘商人**：特殊交易机会
- **未知区域**：特殊事件

## 四、卡牌系统

### 4.1 卡牌基础规则

| 参数 | 数值 |
|------|------|
| 每场战斗初始手牌 | 5张 |
| 每回合抽牌数 | 5张 |
| 初始法力 | 3点 |
| 法力重置 | 每回合重置（部分卡牌提供持续效果） |
| 手牌上限 | 无上限 |
| 弃牌规则 | 回合结束时弃牌堆剩余手牌 |

### 4.2 卡牌分类

#### 按类型分类

| 类型 | 数量占比 | 核心功能 |
|------|----------|----------|
| **攻击牌** | 35% | 造成伤害 |
| **技能牌** | 30% | 获得护甲、抽牌、增益等 |
| **终结技** | 10% | 消耗充能打出的大招 |

#### 按稀有度分类

| 稀有度 | 出现概率 | 颜色标识 | 特点 |
|--------|----------|----------|------|
| **普通** | 50% | 白色 | 稳定、基础效果 |
| **优秀** | 30% | 绿色 | 增强效果或附带次要效果 |
| **稀有** | 15% | 蓝色 | 强力效果或有独特机制 |
| **传说** | 5% | 橙色 | 极强效果或有革命性机制 |

### 4.4 卡牌获取与升级

#### 获取途径
- 战斗胜利后奖励选择
- 商店购买
- 事件奖励
- 精英敌人掉落
- BOSS掉落（专属卡牌、终极技卡牌）

#### 卡牌升级
在篝火或商店可消耗资源升级卡牌，升级效果包括：
- 伤害/治疗数值提升
- 费用降低
- 增加附加效果

```
升级示例：【攻击】→【攻击·极】

基础版：造成6点伤害
升级版：造成9点伤害
```

### 5.2 敌人系统

#### 敌人类型

| 类型 | 特点 | 示例敌人 |
|------|------|----------|
| **普通敌人** | 单一意图，单一机制 | 小妖、狼精、树怪 |
| **精英敌人** | 复杂机制，多阶段 | 虎先锋、牛魔王、红孩儿 |
| **BOSS** | 独特机制，多阶段，召唤物 | 白骨夫人、狮驼岭三魔 |
| **事件敌人** | 非战斗交互 | 土地公（给予祝福/诅咒）|

#### 敌人意图系统
敌人头顶显示下一回合的行动意图：
- ⚔️ 剑：即将攻击（显示伤害数值）
- 🛡️ 盾：即将防御
- 🔮 球：即将施放特殊技能
- ❓ 问号：意图随机
- 💀 骷髅：准备使用高伤害攻击

#### 敌人设计示例

```
【白骨夫人】 - BOSS

第一阶段：幻惑
意图：制造幻影，迷惑玩家
特殊能力：
  - 「画皮」：每3回合召唤一个幻影
  - 「摄魂」：对玩家施加「易伤」
  - 「白骨爪」：造成12点伤害

第二阶段：真身（生命值低于50%）
新增能力：
  - 「骨牢」：封印玩家1张随机手牌
  - 「骨刺阵」：对敌人造成5点伤害（持续叠加）
  - 「夺舍」：若幻影存活，治疗10%最大生命值

第三阶段：白骨精（生命值低于25%）
新增能力：
  - 「白骨森森」：所有敌人攻击力+30%
  - 「涅槃」：每场战斗限一次，死亡后复活并恢复25%生命
```

### 5.3 伤害计算公式

```
最终伤害 = 基础伤害 × (1 + 增伤%) × (1 + 易伤%) × (1 - 减伤%) × (1 - 护甲减免率)

护甲抵消 = min(护甲值, 伤害)
实际承受 = 最终伤害 - 护甲抵消
```

### 5.4 战斗平衡参数

| 参数 | 数值 | 说明 |
|------|------|------|
| 玩家初始生命 | 70 | 可通过道具/卡牌增加 |
| 普通敌人攻击 | 6-12 | 随关卡推进增长 |
| 精英敌人攻击 | 12-20 | 更高风险更高奖励 |
| BOSS攻击 | 15-100 | 多阶段递增 |

| 法力效率（每点法力伤害） | 5-8 | 典型攻击牌数值 |

## 六、成长与进度系统

### 6.1 永久强化

#### 成就系统
完成特定成就解锁永久奖励：

| 成就 | 奖励 | 条件 |
|------|------|------|
| 初窥门径 | 解锁第二角色槽位 | 使用默认角色游玩一局 |
| 18层地狱 | 解锁第三角色槽位 | 任意角色通关第18关 |
| 取得真经 | 解锁第四角色槽位 | 任意角色完成通关 |
| 十全十美 | 解锁无限模式 | 任意角色累计通关10次 |

### 6.2 道具系统

#### 法宝（Magic Weapon）
类似《杀戮尖塔》的遗物系统，战斗开始/获得/触发时生效：

| 遗物 | 效果 | 稀有度 |
|------|------|--------|
| **紧箍圈** | 战斗开始时，若法力≥3，抽1张牌 | 普通 |
| **芭蕉扇** | 攻击时有20%概率造成双倍伤害 | 优秀 |
| **观音玉瓶** | 首次受到致命伤害时，保留1点生命 | 稀有 |
| **风火轮** | 每次攻击有20%概率多触发一次 | 稀有 |
| **镇妖塔** | 战斗开始获得10点护甲，每击败一个敌人+3 | 传说 |

#### 消耗品
单局内使用后消失：

| 道具 | 效果 | 获取方式 |
|------|------|----------|
| **人参果** | 恢复10最大生命值 | 商店、事件 |
| **蟠桃** | 本场战斗生命值+5 | 精英掉落 |
| **玉液琼浆** | 对单个敌人造成50点伤害 本回合只能出3张牌 | 事件 |
| **太乙仙丹** | 力量+10 | 商店 |

### 7.2 动效设计

| 动效类型 | 描述 |
|----------|------|
| **攻击动效** | 卡牌飞出 → 击中 → 目标受击 → 伤害数字飘出，全过程约0.8秒 |
| **技能动效** | 法力汇聚 → 技能释放 → 效果展开，约1.2秒 |
| **终结技动效** | 乌云聚集 → 天雷/火焰降临 → 角色承伤 → 效果触发，约1.5秒 |
| **抽牌动效** | 卡牌从牌组滑出，3D翻转入场，约0.3秒/张 |
| **胜利动效** | 金光普照 + 经文飘落 + 「功德圆满」文字 |
| **死亡动效** | 角色人物消散，灵魂飘起化为光点 |

### 7.3 音效设计

#### 音效风格
以传统民乐为基础，融合现代编曲，塑造「仙侠江湖」的听觉氛围。

| 音效类型 | 乐器选择 | 示例场景 |
|----------|----------|----------|
| **攻击音效** | 琵琶、鼓 | 刀剑出鞘、金属碰撞 |
| **技能音效** | 古筝、笛子 | 仙法施放、法力流动 |
| **终结技音效** | 雷鼓、编钟 | 雷霆万钧、天崩地裂 |
| **背景音乐** | 箫、二胡、琵琶 | 不同区域各有主题音乐 |
| **UI音效** | 木鱼、磬 | 卡牌选中、按钮点击 |

#### 音乐系统
- **主菜单**：悠然笛声，营造归隐山林氛围
- **战斗音乐**：紧凑鼓点 + 琵琶急弦，张力十足
- **BOSS战**：编钟 + 雷鼓，庄严压迫
- **胜利音乐**：琵琶solo + 和声，功德圆满
- **结算动画**：空灵箫声 + 回响效果

## 十、风险评估与应对

### 10.1 核心风险

| 风险类别 | 风险描述 | 应对策略 |
|----------|----------|----------|
| **机制风险** | 天劫卡机制可能过于强力或无用 | 初期简化版本，后续根据数据迭代 |
| **内容风险** | 卡牌池深度不足导致重玩性低 | 优先确保100+卡牌，后续持续更新 |
| **性能风险** | 移动端可能出现卡顿 | 早期多平台测试，优化Draw Call |
| **竞争风险** | 与《杀戮尖塔》直接竞争 | 差异化定位：国风题材+终结技机制 |

### 10.2 质量保障

- **可玩性测试**：每两周一次核心循环测试
- **平衡性测试**：Alpha阶段数据驱动调优
- **用户测试**：Beta阶段收集核心玩家反馈
- **兼容性测试**：覆盖主流设备型号
